## 1. 静夜思 — 李白

**Quiet Night Thoughts**

床前明月光，
疑是地上霜。
举头望明月，
低头思故乡。

### 生词 (Vocabulary)

| 词语 | 拼音 | 意思 |
|------|------|------|
| 静 | jìng | quiet, still |
| 夜 | yè | night |
| 思 | sī | to think, to miss |
| 床 | chuáng | bed |
| 明月 | míng yuè | bright moon |
| 光 | guāng | light |
| 疑 | yí | to suspect, to doubt |
| 霜 | shuāng | frost |
| 举 | jǔ | to raise, to lift |
| 低 | dī | to lower |
| 故乡 | gùxiāng | hometown |

### 诗人介绍

李白（701-762）是唐朝最有名的诗人之一。人们叫他"诗仙"。他一生喜欢旅行，去过中国很多地方。他离开家乡很多年，常常想念家人。这首诗是他在旅行中的一个夜晚写的，那时候他看到月光，心里很想家。

### 诗意理解

一个安静的夜晚，诗人躺在床上，看到月亮的光照在地上。那白白的月光，好像地上有一层霜一样。他抬起头来看天上的明月，月亮又大又圆。然后他低下头，开始想念自己的家乡和家人。月亮让人想家，因为不管你在哪里，你和家人看到的是同一个月亮。这首诗告诉我们，一个人离开家以后，看到美丽的月亮，就会想起远方的家。诗人的心情有一点儿孤单，也有一点儿难过。

### 你知道吗？

这是中国最有名的诗之一。很多中国小朋友学会说话以后，学的第一首诗就是《静夜思》。中秋节的时候，人们常常想起这首诗。

## 2. 春晓 — 孟浩然

**Spring Dawn**

春眠不觉晓，
处处闻啼鸟。
夜来风雨声，
花落知多少。

### 生词 (Vocabulary)

| 词语 | 拼音 | 意思 |
|------|------|------|
| 春 | chūn | spring |
| 眠 | mián | to sleep |
| 觉 | jué | to feel, to notice |
| 晓 | xiǎo | dawn, daybreak |
| 处处 | chùchù | everywhere |
| 闻 | wén | to hear, to smell |
| 啼 | tí | to cry (of birds) |
| 风雨 | fēng yǔ | wind and rain |
| 落 | luò | to fall |

### 诗人介绍

孟浩然（689-740）是唐朝有名的诗人。他很爱大自然，写了很多关于山水和田园的诗。他一生大部分时间住在家乡湖北。他和李白是好朋友。这首诗写的是一个春天的早晨，他刚刚醒来时的感受。

### 诗意理解

春天的时候，天气温暖，人特别容易睡觉。诗人睡得很好，不知不觉天就亮了。他醒来以后，听到外面到处都有鸟叫的声音，非常好听。他想起来了：昨天晚上刮风下雨了，风很大，雨也很大。那么外面的花怎么样了？一定有很多花被风雨吹落了吧？他有一点儿心疼那些花。这首诗写出了春天的美好，也写出了诗人对花儿的关心。美好的东西不能永远留住，这让人有一点儿难过。

### 你知道吗？

这首诗只有二十个字，但是用了听觉（鸟叫声、风雨声）来写春天，让读者好像真的在春天的早晨一样。中国人常用"春眠不觉晓"来说春天很容易让人困。

## 3. 登鹳雀楼 — 王之涣

**Climbing the Stork Tower**

白日依山尽，
黄河入海流。
欲穷千里目，
更上一层楼。

### 生词 (Vocabulary)

| 词语 | 拼音 | 意思 |
|------|------|------|
| 登 | dēng | to climb |
| 楼 | lóu | tower, building |
| 白日 | bái rì | sun |
| 依 | yī | to lean on, along |
| 尽 | jìn | to end, to exhaust |
| 黄河 | Huáng Hé | Yellow River |
| 入 | rù | to enter, to flow into |
| 海 | hǎi | sea |
| 欲 | yù | to want, to wish |
| 穷 | qióng | to exhaust, to reach the end |
| 千里 | qiān lǐ | a thousand miles (very far) |
| 更 | gèng | even more |
| 层 | céng | layer, floor |

### 诗人介绍

王之涣（688-742）是唐朝的诗人。他写的诗不多，但是很多都非常有名。鹳雀楼在山西省，是一座很高的楼。王之涣有一天去那里玩，爬上了这座楼，看到了非常美的风景，就写了这首诗。

### 诗意理解

诗人站在高高的鹳雀楼上面，看到太阳慢慢地落到远处的山后面去了。他还看到黄河的水从远方来，向着大海的方向流去。这个风景太美了！他想看得更远一些，想看到千里以外的风景。怎么办呢？他就再上一层楼吧！站得更高，就能看得更远。这首诗不只是说风景，还告诉我们一个道理：如果你想有更大的进步，你就需要更加努力，站到更高的地方。

### 你知道吗？

"欲穷千里目，更上一层楼"是中国人最常用的名句之一。老师和父母常常用这句话鼓励孩子努力学习。它的意思是：想做到更好，就要继续努力。

## 4. 悯农 — 李绅

**Pity the Farmers**

锄禾日当午，
汗滴禾下土。
谁知盘中餐，
粒粒皆辛苦。

### 生词 (Vocabulary)

| 词语 | 拼音 | 意思 |
|------|------|------|
| 悯 | mǐn | to pity, to sympathize |
| 农 | nóng | farmer |
| 锄 | chú | to hoe |
| 禾 | hé | grain, crops |
| 汗 | hàn | sweat |
| 滴 | dī | to drip |
| 土 | tǔ | soil, earth |
| 盘 | pán | plate, dish |
| 餐 | cān | meal, food |
| 粒 | lì | grain (measure word) |
| 皆 | jiē | all |
| 辛苦 | xīnkǔ | hard, toilsome |

### 诗人介绍

李绅（772-846）是唐朝的诗人和官员。他小时候家里很穷，所以他知道农民的生活很辛苦。后来他当了官，但是他没有忘记农民。他看到农民在太阳下面辛苦地工作，心里很感动，就写了这首诗。

### 诗意理解

中午的太阳非常热，但是农民还在田里工作。他们用锄头锄地上的草，汗水一滴一滴地落到地里。他们的衣服都湿了，但是不能休息，因为庄稼需要照顾。诗人问大家：你们吃饭的时候，有没有想过盘子里的米饭是怎么来的？每一粒米饭都是农民辛辛苦苦种出来的啊！我们不应该浪费粮食。这首诗让我们知道要尊重农民的劳动，珍惜每一粒粮食。

### 你知道吗？

在中国，父母常常用这首诗教育孩子不要浪费食物。如果小朋友不好好吃饭或者浪费米饭，大人就会说："谁知盘中餐，粒粒皆辛苦。"这首诗每个中国人都会背。

## 5. 早发白帝城 — 李白

**Departing from Baidi City at Dawn**

朝辞白帝彩云间，
千里江陵一日还。
两岸猿声啼不住，
轻舟已过万重山。

### 生词 (Vocabulary)

| 词语 | 拼音 | 意思 |
|------|------|------|
| 朝 | zhāo | morning, dawn |
| 辞 | cí | to bid farewell, to leave |
| 彩云 | cǎi yún | colorful clouds |
| 千里 | qiān lǐ | a thousand miles |
| 江陵 | Jiānglíng | Jiangling (a city) |
| 还 | huán | to return |
| 两岸 | liǎng àn | both banks |
| 猿 | yuán | monkey, ape |
| 轻舟 | qīng zhōu | light boat |
| 已 | yǐ | already |
| 过 | guò | to pass |
| 万重山 | wàn chóng shān | ten thousand layers of mountains |

### 诗人介绍

李白写这首诗的时候，他的心情非常高兴。那时候他遇到了麻烦，被关进了监狱。后来皇帝放了他，他非常开心。他从白帝城坐船出发，沿着长江回家。心情好的时候，他觉得船走得特别快，风景特别美，就写下了这首诗。

### 诗意理解

早晨，诗人离开了白帝城。白帝城在很高的山上，早上的时候，山上面有很多彩色的云。他坐上一条小船，沿着长江往东走。船走得非常非常快，一千里的路，一天就到了江陵！船的两边是高高的山，山上有猴子一直在叫，声音一直没停。但是船太快了，猴子的叫声还没听完，小船已经过了一座又一座的山。诗人的心情非常愉快，因为他自由了。整首诗让人感觉到速度和快乐。

### 你知道吗？

白帝城在今天的重庆市，长江三峡的入口。三峡的水流很快，所以船顺着水走的时候速度非常快。现在三峡是中国最有名的旅游景点之一。

## 6. 相思 — 王维

**Longing**

红豆生南国，
春来发几枝。
愿君多采撷，
此物最相思。

### 生词 (Vocabulary)

| 词语 | 拼音 | 意思 |
|------|------|------|
| 相思 | xiāngsī | to miss someone, longing |
| 红豆 | hóng dòu | red bean |
| 生 | shēng | to grow |
| 南国 | nán guó | the south, southern land |
| 发 | fā | to sprout, to grow |
| 枝 | zhī | branch |
| 愿 | yuàn | to wish, to hope |
| 君 | jūn | you (respectful) |
| 采撷 | cǎi xié | to pick, to gather |
| 此 | cǐ | this |
| 物 | wù | thing, object |

### 诗人介绍

王维（701-761）是唐朝非常有名的诗人和画家。人们叫他"诗佛"。他的诗写得像画一样美。他有一个好朋友要去很远的地方，他非常想念朋友。在中国文化里，红豆代表想念一个人。王维用红豆来表达对朋友的思念，写下了这首诗。

### 诗意理解

在中国南方，有一种树会长出红色的小豆子，非常漂亮。春天来了，红豆树又长出了新的枝条。诗人对他的朋友说：请你多采一些红豆吧，因为红豆最能代表想念一个人的心情。诗人没有直接说"我想你"，而是用红豆来表达他的感情。这样的写法更加温暖，更加有感情。不管是想念朋友还是想念爱人，看到红豆，就会想起远方的那个人。

### 你知道吗？

在中国文化里，红豆是"想念"和"爱情"的象征。现在很多中国人送红豆给自己喜欢的人，表示"我想你"。红豆也是一种常见的食物，中国人用它做甜点。

## 7. 清明 — 杜牧

**Qingming Festival**

清明时节雨纷纷，
路上行人欲断魂。
借问酒家何处有，
牧童遥指杏花村。

### 生词 (Vocabulary)

| 词语 | 拼音 | 意思 |
|------|------|------|
| 清明 | Qīngmíng | Qingming Festival (Tomb-Sweeping Day) |
| 时节 | shíjié | season, time |
| 纷纷 | fēnfēn | one after another (of rain) |
| 行人 | xíngrén | traveler, pedestrian |
| 断魂 | duàn hún | heartbroken, extremely sad |
| 借问 | jiè wèn | may I ask |
| 酒家 | jiǔ jiā | wine shop, tavern |
| 何处 | héchù | where |
| 牧童 | mùtóng | shepherd boy |
| 遥 | yáo | far, distant |
| 指 | zhǐ | to point |
| 杏花 | xìng huā | apricot blossom |

### 诗人介绍

杜牧（803-852）是唐朝后期有名的诗人。人们叫他"小杜"，因为还有一个更有名的诗人叫杜甫，被叫做"老杜"。杜牧的诗写得很有感情。清明节是中国人去扫墓的日子。那一天，杜牧一个人在路上走，天下着小雨，他心情不好，就写了这首诗。

### 诗意理解

清明节到了，天上下着小雨，雨一直不停。路上有很多人，他们要去扫墓，想念已经不在的家人。大家的心情都很难过。诗人也是一样，他走在路上，心里很不舒服。他又冷又难过，想找一个地方喝一杯酒，让自己暖和一下，心情也好一点儿。他问路边的一个放牛的小孩："哪里有卖酒的地方？"那个小孩用手指向远处，那里有一个小村子，村子里开满了杏花。诗人看到那个美丽的村子，心情好了一些。

### 你知道吗？

清明节在每年四月初，是中国最重要的传统节日之一。这一天，中国人会去扫墓，纪念去世的家人。清明节的时候，中国很多地方常常下雨，所以人们总会想起这首诗。

## 8. 江雪 — 柳宗元

**River Snow**

千山鸟飞绝，
万径人踪灭。
孤舟蓑笠翁，
独钓寒江雪。

### 生词 (Vocabulary)

| 词语 | 拼音 | 意思 |
|------|------|------|
| 江 | jiāng | river |
| 雪 | xuě | snow |
| 千山 | qiān shān | a thousand mountains |
| 绝 | jué | to disappear completely |
| 万 | wàn | ten thousand |
| 径 | jìng | path, trail |
| 踪 | zōng | trace, footprint |
| 灭 | miè | to vanish, to disappear |
| 孤 | gū | lonely, solitary |
| 舟 | zhōu | boat |
| 蓑笠 | suō lì | straw raincoat and hat |
| 翁 | wēng | old man |
| 独 | dú | alone |
| 钓 | diào | to fish |
| 寒 | hán | cold |

### 诗人介绍

柳宗元（773-819）是唐朝有名的文学家。他因为政治原因被送到了很远的地方，不能回家。他一个人在那里生活，心情很孤独。他用一个在雪中钓鱼的老人来写自己的感受。这首诗是中国最有名的写雪的诗之一。

### 诗意理解

冬天，大雪下了很久。你看那远处的山，一座一座，全是白色的雪，天上没有一只鸟在飞。你看那地上的路，每一条路上都没有人走过的脚印，人都不见了。整个世界安静得不得了。在这个白茫茫的世界里，有一条小船。船上坐着一个老人，他穿着蓑衣，戴着斗笠，一个人在冰冷的江上钓鱼。天这么冷，到处都是雪，可是这个老人不怕冷，也不觉得孤单。他就是要一个人安安静静地待着。诗人写这个老人，其实是在写自己。

### 你知道吗？

这首诗只有二十个字，但是画面感非常强。很多中国画家都画过这首诗描写的画面：白茫茫的雪中，一个老人一个人钓鱼。这首诗每一句里都有一个"大"的数字（千、万），但是写的是"空"和"没有"，这是很高明的写法。

## 9. 回乡偶书 — 贺知章

**Returning Home**

少小离家老大回，
乡音无改鬓毛衰。
儿童相见不相识，
笑问客从何处来。

### 生词 (Vocabulary)

| 词语 | 拼音 | 意思 |
|------|------|------|
| 回乡 | huí xiāng | to return to hometown |
| 偶 | ǒu | by chance, occasional |
| 少小 | shào xiǎo | young, in youth |
| 老大 | lǎo dà | old, in old age |
| 乡音 | xiāng yīn | local accent, dialect |
| 无 | wú | without, not |
| 改 | gǎi | to change |
| 鬓毛 | bìn máo | hair at the temples |
| 衰 | shuāi | to decline, thin |
| 儿童 | értóng | children |
| 相识 | xiāng shí | to recognize, to know |
| 客 | kè | guest, stranger |

### 诗人介绍

贺知章（659-744）是唐朝的诗人和官员。他年轻的时候就离开了家乡浙江去做官，一直在外面工作了五十多年。他八十多岁的时候才回到家乡。回到家以后，他发现家乡变了很多，没有人认识他了。他心里又高兴又感慨，就写下了这首诗。

### 诗意理解

诗人小时候就离开了家乡，到了老的时候才回来。他在外面生活了很多很多年，虽然他说话的口音没有变，还是家乡话，但是他的头发已经白了，样子完全变了。他走进村子里，村子里的小孩子们看到他，都不认识他。孩子们笑着问他："老爷爷，您是从哪里来的呀？"这让诗人心里很感慨：这里是我的家啊，可是没有人认识我了。时间过得太快了，一切都变了。这首诗让人感觉到时间的力量，也让人想到"家"的意义。

### 你知道吗？

这首诗写的感情每个人都能理解。很多中国人在外面工作很多年，回到家乡的时候也有和诗人一样的感受。所以"少小离家老大回"成了表达这种心情的经典名句。

## 10. 望庐山瀑布 — 李白

**Watching the Waterfall at Mount Lu**

日照香炉生紫烟，
遥看瀑布挂前川。
飞流直下三千尺，
疑是银河落九天。

### 生词 (Vocabulary)

| 词语 | 拼音 | 意思 |
|------|------|------|
| 望 | wàng | to gaze at, to look at |
| 庐山 | Lú Shān | Mount Lu |
| 瀑布 | pùbù | waterfall |
| 日照 | rì zhào | sunshine, sunlight shines on |
| 香炉 | xiāng lú | incense burner (a mountain peak) |
| 紫烟 | zǐ yān | purple mist |
| 遥 | yáo | from far away |
| 挂 | guà | to hang |
| 川 | chuān | river, stream |
| 飞流 | fēi liú | flying stream |
| 直下 | zhí xià | straight down |
| 三千尺 | sān qiān chǐ | three thousand feet |
| 银河 | Yínhé | Milky Way |
| 九天 | jiǔ tiān | the highest heavens |

### 诗人介绍

李白非常喜欢旅行和自然风景。有一次他去了江西的庐山，看到了那里的大瀑布。庐山的瀑布非常壮观，水从很高的山上流下来。李白被这个美丽的风景打动了，用他最擅长的夸张写法，写下了这首有名的诗。

### 诗意理解

太阳照在庐山的香炉峰上面，山上升起了紫色的云雾，非常美丽。诗人站在远处，看到一条瀑布挂在山上，就像一条白色的布挂在那里一样。瀑布的水飞快地从上面直接流下来，有三千尺那么高！这水流得太壮观了，诗人觉得这不像是人间的瀑布，好像是天上的银河从最高的天上落下来的！李白用了非常大胆的想象——把瀑布比作银河，让我们感受到了大自然的壮观和伟大。

### 你知道吗？

"飞流直下三千尺，疑是银河落九天"是中国文学里最有名的夸张句之一。李白特别喜欢用夸张的手法写诗，所以人们叫他"诗仙"。庐山在今天的江西省，现在是中国的5A级旅游景区和世界文化遗产。
