# 中国古诗 (Chinese Classical Poems)

**HSK Level 2**

有：原文、生词、意思。

---

## 1. 静夜思 — 李白

**原文：**

床前明月光，
疑是地上霜。
举头望明月，
低头思故乡。

**生词：** 疑 = think, 霜 = frost, 望 = gaze, 思 = miss

**意思：**

一个不亮的时候，他一个人在外面住。他睡不着。他看到亮的光照在床前面的地上。他抬起头看天上亮的，然后低下头，想自己的家。亮的还那个亮的，但他的家很远很远。

这首诗说的一个东西——想家。

---

## 2. 春晓 — 孟浩然

**原文：**

春眠不觉晓，
处处闻啼鸟。
夜来风雨声，
花落知多少。

**生词：** 眠 = sleep, 晓 = dawn, 闻 = hear, 啼 = chirp, 落 = fall

**意思：**

春天的时候，睡觉很好。天亮了，我还不知道。鸟在喊，我醒了——到处都有鸟在喊。昨天的不亮时间有很大的风和雨。那些花落了不少吧？落了多少？我不知道。

他醒了，不先想到自己，而想到了花。

## 3. 登鹳雀楼 — 王之涣

**原文：**

白日依山尽，
黄河入海流。
欲穷千里目，
更上一层楼。

**生词：** 依 = lean, 尽 = end, 穷 = exhaust, 目 = eyes, 层 = floor

**意思：**

太阳慢慢地下去了。黄河很大，向着大海流去。我想看到更远的地方，所以我再上一层楼。站得更高，就能看得更远。

这首诗还说：你想看到更多，知道更多，就让自己变得更高更好。

## 4. 悯农 — 李绅

**原文：**

锄禾日当午，
汗滴禾下土。
谁知盘中餐，
粒粒皆辛苦。

**生词：** 锄 = hoe, 禾 = grain, 汗 = sweat, 滴 = drip

**意思：**

中午的时候，太阳最热。做活的人在外面工作。我们吃饭的时候，谁想了这碗饭怎么来的？一个一个饭都做活的人很辛苦地做出来的。

在中国，小孩吃饭剩了很多的时候，大人会说这首诗。

## 5. 早发白帝城 — 李白

**原文：**

朝辞白帝彩云间，
千里江陵一日还。
两岸猿声啼不住，
轻舟已过万重山。

**生词：** 朝 = morning, 辞 = leave, 猿 = monkey, 舟 = boat

**意思：**

早上，我离开了一个很高的地方。那里旁都彩色的云。船很快——一千里的路，一天就到了！两面的猴一直在喊。我的小船已经了一个又一个的大的地方。

这首诗让人觉得很快、很高兴。

## 6. 相思 — 王维

**原文：**

红豆生南国，
春来发几枝。
愿君多采撷，
此物最相思。

**生词：** 红豆 = red beans, 愿 = wish, 君 = you, 采 = pick

**意思：**

南面有红豆的东西，春天的时候长出新的。你多拿一些红豆吧。因为红豆的意思想念一个人。

在中国，红豆"想念"的意思。

## 7. 清明 — 杜牧

**原文：**

清明时节雨纷纷，
路上行人欲断魂。
借问酒家何处有，
牧童遥指杏花村。

**生词：** 清明 = Qingming, 纷纷 = in succession, 断魂 = heartbroken

**意思：**

清明节这天，雨一直下。路上走着的人心里都不开心。我问别人："哪里有喝酒的地方？"一个小孩用手指了指远处。

他想找一个地方喝酒，不因为高兴，而因为心里太不好了。

## 8. 江雪 — 柳宗元

**原文：**

千山鸟飞绝，
万径人踪灭。
孤舟蓑笠翁，
独钓寒江雪。

**生词：** 绝 = gone, 径 = path, 踪 = trace, 孤 = alone, 钓 = fish

**意思：**

所有的大的地方上，一只鸟没有。所有的路上，一个人没有。只有一个老人，穿着雨的衣，一个人坐在一条小船上。天很冷，在下雪。他就这样一个人在那里。

这首诗只有二十个字，但我们能觉得那里很安静，只有一个人。什么都没有，只有雪和一个人。

## 9. 回乡偶书 — 贺知章

**原文：**

少小离家老大回，
乡音无改鬓毛衰。
儿童相见不相识，
笑问客从何处来。

**生词：** 乡音 = accent, 鬓 = temple hair, 衰 = thin

**意思：**

我小的时候就离开了家。现在我老了，回来了。我说得还家那里的样，但我的头发变了。那里的小孩看到我，不认识我。他们问我："你从哪里来的？"

他们不知道，我不外面的人——这里就我的家。但我离开了这么长时间，没有人认识我了。

## 10. 望庐山瀑布 — 李白

**原文：**

日照香炉生紫烟，
遥看瀑布挂前川。
飞流直下三千尺，
疑是银河落九天。

**生词：** 瀑布 = waterfall, 烟 = mist, 银河 = Milky Way

**意思：**

太阳照在上面，上面的雾变成了紫色。远远地看那个瀑布，像挂在前面一样。很高的地方飞下来，有三千尺那么高！我想那天上的银河掉下来了。

他最喜欢说得很大。瀑布没有三千尺高，不银河。但他这么说，我们就能觉得那个瀑布很大、很美了。
