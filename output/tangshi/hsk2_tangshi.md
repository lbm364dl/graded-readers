# 中国古诗 (Chinese Classical Poems)

**HSK Level 2**

每首诗包括：原文、生词、故事背景、白话翻译。

---

## 1. 静夜思 — 李白

**原文：**

床前明月光，
疑是地上霜。
举头望明月，
低头思故乡。

**生词：** 疑 = suspect/think, 霜 = frost, 举 = raise, 望 = gaze, 思 = miss/think of

**诗人：** 李白是中国最有名的诗人，人们叫他"诗仙"。他喜欢喝酒，喜欢旅行，写了很多很多诗。

**白话意思：**

一个晚上，李白一个人在外面住。他睡不着觉。他看到月亮的光照在床前面的地上，以为是霜。他抬起头看月亮，然后低下头，开始想自己的家乡。月亮还是那个月亮，可是他的家很远很远。

这首诗是中国人最早学的一首诗。几乎每个中国小孩都会背。它说的是一个很简单的感情——想家。

## 2. 春晓 — 孟浩然

**原文：**

春眠不觉晓，
处处闻啼鸟。
夜来风雨声，
花落知多少。

**生词：** 眠 = sleep, 晓 = dawn, 闻 = hear, 啼 = chirp/cry, 落 = fall

**诗人：** 孟浩然是唐朝的诗人。他住在山里，不当官，喜欢自然。

**白话意思：**

春天的时候，睡觉特别舒服。早上天亮了，我还不知道。我是被鸟的叫声叫醒的——到处都有鸟在叫。我忽然想起来，昨天晚上有很大的风和雨。那些花一定落了不少吧？可是到底落了多少，我也不知道。

诗人醒来，不是先想到自己，而是想到了花。这说明他是一个很喜欢自然的人。

## 3. 登鹳雀楼 — 王之涣

**原文：**

白日依山尽，
黄河入海流。
欲穷千里目，
更上一层楼。

**生词：** 依 = lean against, 尽 = end, 穷 = exhaust, 目 = eyes, 层 = floor/level

**诗人：** 王之涣是唐朝诗人。鹳雀楼是一座有名的楼，在黄河旁边。

**白话意思：**

太阳靠着山，慢慢地下去了。黄河很大，向着大海流去。我想要看到更远的地方，所以我要再上一层楼。站得更高，就能看得更远。

这首诗不只是在说看风景。它还告诉我们一个道理：如果你想看到更多，了解更多，就要努力让自己变得更高更好。

## 4. 悯农 — 李绅

**原文：**

锄禾日当午，
汗滴禾下土。
谁知盘中餐，
粒粒皆辛苦。

**生词：** 锄 = hoe, 禾 = grain, 汗 = sweat, 滴 = drip, 粒 = grain (counter)

**白话意思：**

中午的时候，太阳最热。农民在田里干活，汗水一滴一滴地落在土上。我们吃饭的时候，谁想过这碗饭是怎么来的？每一粒米饭都是农民辛辛苦苦种出来的。

在中国，如果小孩吃饭剩了很多，大人常常会说这首诗。

## 5. 早发白帝城 — 李白

**原文：**

朝辞白帝彩云间，
千里江陵一日还。
两岸猿声啼不住，
轻舟已过万重山。

**生词：** 朝 = morning, 辞 = leave, 猿 = ape/monkey, 舟 = boat, 重 = layer

**诗人故事：** 李白被皇帝关起来了，后来被放了。他非常高兴，坐船回家。因为高兴，他觉得什么都快，什么都好。

**白话意思：**

早上，我离开了白帝城。那里在很高的山上，周围都是彩色的云。船非常快——一千里的路，一天就到了！两岸的猴子一直在叫，声音不停。可是我的小船已经过了一座又一座的山。

这首诗让人感觉到速度和快乐。因为李白刚刚被放出来，所以他觉得世界特别美好。

## 6. 相思 — 王维

**原文：**

红豆生南国，
春来发几枝。
愿君多采撷，
此物最相思。

**生词：** 红豆 = red beans, 愿 = wish, 君 = you (polite), 采 = pick, 相思 = longing/missing

**白话意思：**

南方有红豆树，春天的时候长出新的枝条。希望你多摘一些红豆。因为红豆代表想念一个人。

王维写这首诗的时候，是在想一个远方的朋友。在中国文化里，红豆是"想念"的意思。现在很多甜品里面放红豆，年轻人会想到这首诗。

## 7. 清明 — 杜牧

**原文：**

清明时节雨纷纷，
路上行人欲断魂。
借问酒家何处有，
牧童遥指杏花村。

**生词：** 清明 = Qingming Festival, 纷纷 = in succession, 断魂 = heartbroken, 牧童 = shepherd boy, 遥 = far away

**背景：** 清明节是中国人去看已经去世的家人的日子。人们去扫墓，心情难过。

**白话意思：**

清明节这天，雨一直下，下个不停。路上走着的人心里都很难过。我问别人："哪里有卖酒的地方？"一个放羊的小孩用手指了指远处，那边有一个村子，叫杏花村。

诗人想找个地方喝酒，不是因为高兴，而是因为太难过了。

## 8. 江雪 — 柳宗元

**原文：**

千山鸟飞绝，
万径人踪灭。
孤舟蓑笠翁，
独钓寒江雪。

**生词：** 绝 = disappeared, 径 = path, 踪 = trace, 孤 = alone, 独 = solitary, 钓 = fish

**诗人故事：** 柳宗元当过官，后来被赶到很远的地方。他很孤独。

**白话意思：**

所有的山上，一只鸟也没有。所有的路上，一个人的脚印也没有。只有一个老人，穿着雨衣，戴着帽子，一个人坐在一条小船上。天很冷，在下雪。他就这样一个人在江上钓鱼。

这首诗只有二十个字，可是我们能感觉到那种安静和孤独。什么都没有，只有雪和一个人。这个人就是诗人自己。

## 9. 回乡偶书 — 贺知章

**原文：**

少小离家老大回，
乡音无改鬓毛衰。
儿童相见不相识，
笑问客从何处来。

**生词：** 少小 = young, 乡音 = hometown accent, 鬓 = temples (hair), 衰 = thin/declining, 识 = recognize

**白话意思：**

我小时候就离开了家乡。现在我老了，终于回来了。我说话的口音没有变，可是我的头发已经白了。村子里的小孩看到我，不认识我。他们笑着问我："你是从哪里来的客人？"

他们不知道，我不是客人——这里就是我的家。可是我离开了这么久，在家乡变成了陌生人。这种感觉很难过。

## 10. 望庐山瀑布 — 李白

**原文：**

日照香炉生紫烟，
遥看瀑布挂前川。
飞流直下三千尺，
疑是银河落九天。

**生词：** 瀑布 = waterfall, 烟 = smoke/mist, 飞 = fly, 银河 = Milky Way

**白话意思：**

太阳照在山上，山上的雾气变成了紫色。远远地看那个瀑布，好像挂在山前面一样。水从很高的地方飞下来，有三千尺那么高！我以为那是天上的银河掉下来了。

李白最喜欢用夸张的说法。瀑布当然没有三千尺高，更不是银河。可是他这么说，我们就能感觉到那个瀑布有多么大、多么美了。
