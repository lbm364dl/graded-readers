# 宋词 (Song Dynasty Ci Poetry)

**HSK Level 2**

每首词包括：原文、生词、词人故事、白话意思。

---

## 1. 水调歌头 — 苏轼

**原文：**

明月几时有？把酒问青天。
不知天上宫阙，今夕是何年。
我欲乘风归去，又恐琼楼玉宇，高处不胜寒。
起舞弄清影，何似在人间。

转朱阁，低绮户，照无眠。
不应有恨，何事长向别时圆？
人有悲欢离合，月有阴晴圆缺，此事古难全。
但愿人长久，千里共婵娟。

**生词：** 月亮 = moon, 酒 = wine, 天上 = in the sky, 影子 = shadow, 离开 = to part

**词人：** 苏轼是宋朝最有名的词人。人们叫他苏东坡。他很聪明，会写词，也会写文章和画画。

**白话意思：**

中秋节的晚上，苏轼一个人在外面。他喝着酒，看着天上又大又圆的月亮。他问月亮："你是什么时候开始有的？"他想飞到天上去，可是天上太冷了。他想，还是在人间好。月亮照着他，他睡不着。他想起了弟弟。弟弟在很远的地方，已经好几年没有见面了。他问月亮："为什么你总是在人们分开的时候才圆呢？"

后来他想通了：人有高兴、难过、分开、在一起的时候，月亮也有圆有缺。这是自古以来就这样的，没有人能改变。他希望自己和弟弟都能一直好好的，虽然离得很远，但是看到的是同一个月亮。

"但愿人长久，千里共婵娟"是中国人最喜欢说的话之一。每年中秋节，人们都会想起这句话。

## 2. 如梦令 — 李清照

**原文：**

常记溪亭日暮，沉醉不知归路。
兴尽晚回舟，误入藕花深处。
争渡，争渡，惊起一滩鸥鹭。

**生词：** 船 = boat, 花 = flower, 鸟 = bird, 路 = road

**词人：** 李清照是宋朝最有名的女词人。她很聪明，从小就喜欢读书写词。

**白话意思：**

李清照常常想起年轻时候的一天。那天，她和朋友在小河旁边的小亭子里喝酒、聊天。太阳快下山了，她喝了很多酒，太开心了，忘记了回家的路。她坐着小船回去，可是走错了方向，到了一片花中间。她想快一点出去，就使劲划船。船动得很快，把水边的很多白鸟都吓飞了。

这首词写的是一个非常快乐的日子。它很短，但是让人能看到那个画面：太阳快下山了，一个年轻女孩在花中间划船，白鸟飞起来。多么美好！

## 3. 青玉案 — 辛弃疾

**原文：**

东风夜放花千树，更吹落、星如雨。
宝马雕车香满路。凤箫声动，玉壶光转，一夜鱼龙舞。

蛾儿雪柳黄金缕，笑语盈盈暗香去。
众里寻他千百度，蓦然回首，那人却在，灯火阑珊处。

**生词：** 灯 = lamp, 夜晚 = night, 星星 = star, 美女 = beautiful woman

**词人：** 辛弃疾是宋朝的大词人，也是一个打仗的人。他又会写词又会打仗，非常了不起。

**白话意思：**

这首词写的是元宵节的晚上。过节了，到处都是花灯，非常好看。就好像很多花开在了树上。天上的星星好像下雨一样。路上有很多好看的马和车，到处都很香。人们在唱歌跳舞，玩了一整个晚上。

很多女孩走过去了，她们穿得很好看，说着笑着，带着香味。可是词人不看她们，他在找一个人。他找了很久很久，找了千百次，都没有找到。后来他忽然回头一看，那个人就在后面，站在灯光不太亮的地方，一个人安安静静地站着。

很多人觉得这首词不只是在说找人。它也在说：有时候我们找的东西，就在不远的地方。

## 4. 念奴娇 — 苏轼

**原文：**

大江东去，浪淘尽、千古风流人物。
故垒西边，人道是、三国周瑜赤壁。
乱石穿空，惊涛拍岸，卷起千堆雪。
江山如画，一时多少豪杰。

遥想公瑾当年，小乔初嫁了，雄姿英发。
羽扇纶巾，谈笑间、樯橹灰飞烟灭。
故国神游，多情应笑我，早生华发。
人生如梦，一尊还酹江月。

**生词：** 大江 = great river, 浪 = waves, 英雄 = hero, 以前 = before

**词人故事：** 苏轼写这首词的时候被送到了很远的地方。他到了赤壁，这里是以前三国时候打仗的地方。他想到了以前的英雄周瑜。

**白话意思：**

大江的水向东流去，大浪一个接一个。很多很多年过去了，以前那些有名的人都不在了。这个地方是赤壁。三国的时候，周瑜在这里打了一场非常有名的仗。你看，石头非常高，大浪打在岸上，好像下雪一样。这里的山和水好像一幅画。那个时候有多少英雄啊！

想想周瑜那个时候，他很年轻，又帅又有本事。他一边说笑一边就把敌人打败了。可是现在，这些人都不在了。苏轼看着这里的山水，想着以前的事情，觉得自己的头发都白了。他说：人生就好像做了一个梦。他用酒敬了月亮。

## 5. 虞美人 — 李煜

**原文：**

春花秋月何时了？往事知多少。
小楼昨夜又东风，故国不堪回首月明中。

雕栏玉砌应犹在，只是朱颜改。
问君能有几多愁？恰似一江春水向东流。

**生词：** 春天 = spring, 月亮 = moon, 水 = water, 愁 = sorrow

**词人故事：** 李煜以前是南唐的国王。后来，他的国家被宋朝打败了。他被带到了宋朝的首都，变成了一个没有自由的人。他很难过，就写了这首词。

**白话意思：**

春天的花，秋天的月亮，什么时候才能结束？以前的事情我还记得太多太多。昨天晚上，春风又来了。在月光下面，我想起了我的国家，心里很痛。

那些好看的房子应该还在吧？可是住在那里的人已经老了。你问我有多难过？我的难过就好像一条大江的水一样多，一直一直向东流。

这首词是中国最有名的"难过的词"。李煜用大江的水来说他的难过，让人觉得那种难过没有头。

## 6. 雨霖铃 — 柳永

**原文：**

寒蝉凄切，对长亭晚，骤雨初歇。
都门帐饮无绪，留恋处、兰舟催发。
执手相看泪眼，竟无语凝噎。
念去去、千里烟波，暮霭沉沉楚天阔。

多情自古伤离别，更那堪、冷落清秋节。
今宵酒醒何处？杨柳岸、晓风残月。
此去经年，应是良辰好景虚设。
便纵有千种风情，更与何人说？

**生词：** 离开 = to leave, 手 = hand, 眼睛 = eyes, 秋天 = autumn, 柳树 = willow

**词人：** 柳永是宋朝有名的词人。他一生不太顺利，常常在外面旅行。他写了很多很好听的词。

**白话意思：**

秋天了，天很冷。他要离开了。他们在一个亭子旁边喝告别的酒。船已经准备好了，催他快走。他拉着朋友的手，两个人看着对方的眼睛，都流泪了，可是一句话也说不出来。他想，走了以后，前面是千里的路，天很大很远。

从古到今，离开总是让人难过的。今天晚上我喝了酒，明天早上醒来会在哪里呢？也许在一条河边，旁边有柳树，风很冷，月亮只有一半。以后就算看到好的风景，好的天气，没有人一起看，有什么用呢？好的东西想和谁说呢？

这首词是中国最有名的"送别词"。人们离开的时候，常常会想起它。

## 7. 满江红 — 岳飞

**原文：**

怒发冲冠，凭栏处、潇潇雨歇。
抬望眼，仰天长啸，壮怀激烈。
三十功名尘与土，八千里路云和月。
莫等闲、白了少年头，空悲切。

靖康耻，犹未雪；臣子恨，何时灭。
驾长车，踏破贺兰山缺。
壮志饥餐胡虏肉，笑谈渴饮匈奴血。
待从头、收拾旧山河，朝天阙。

**生词：** 生气 = angry, 年轻 = young, 国家 = country, 头发 = hair

**词人故事：** 岳飞是宋朝最有名的大将军。他非常爱自己的国家。那时候北方的人打进来了，把宋朝的首都占了。岳飞打仗很厉害，想把国家拿回来。

**白话意思：**

我很生气！我的头发都要竖起来了。我站在那里，看着外面的雨，抬头向天大叫。我的心里有很多很多想做的事。

我今年三十岁了。我走了八千里的路，看了很多云和月亮。可是到现在还没有做成什么大事。不要等了！不要等到年轻人的头发都白了，那时候再难过就太晚了。

我们的首都被敌人占了，这是多大的耻辱！我要骑着马去打败他们。我要把我们的山和河都拿回来！

这首词非常有力量。每个中国人读了都会觉得心里很热。

## 8. 钗头凤 — 陆游

**原文：**

红酥手，黄縢酒，满城春色宫墙柳。
东风恶，欢情薄。一怀愁绪，几年离索。
错、错、错。

春如旧，人空瘦，泪痕红浥鲛绡透。
桃花落，闲池阁。山盟虽在，锦书难托。
莫、莫、莫。

**生词：** 春天 = spring, 酒 = wine, 错 = wrong, 哭 = to cry

**词人故事：** 陆游和他的妻子很相爱。可是陆游的妈妈不喜欢她，让他们分开了。后来有一天，陆游在一个公园里忽然看到了她。他很难过，就在墙上写了这首词。

**白话意思：**

春天到了。到处都很好看，柳树是绿色的。我看到了她。她拿着一杯酒。以前我们一起看春天，一起喝酒，那时候多好啊。可是，命运不好，我们分开了，到现在已经好几年了。这是错的，全都错了。

她还是她，可是瘦了很多。她的脸上有哭过的痕迹。桃花落了。以前我们说过的话还记得，可是我们再也不能在一起了。算了吧，算了吧，不要再想了。

这首词让很多人看了都会流泪。

## 9. 声声慢 — 李清照

**原文：**

寻寻觅觅，冷冷清清，凄凄惨惨戚戚。
乍暖还寒时候，最难将息。
三杯两盏淡酒，怎敌他、晚来风急？
雁过也，正伤心，却是旧时相识。

满地黄花堆积，憔悴损、如今有谁堪摘？
守着窗儿，独自怎生得黑？
梧桐更兼细雨，到黄昏、点点滴滴。
这次第，怎一个愁字了得！

**生词：** 找 = to look for, 冷 = cold, 一个人 = alone, 雨 = rain

**词人故事：** 李清照的丈夫去世了。她一个人在南方。她以前的家在北方，可是北方被敌人占了，回不去了。她非常孤独，非常难过。

**白话意思：**

我找啊找，什么都找不到。到处都冷冷清清的。天有时暖有时冷，我的身体受不了。我喝了两三杯酒，可是没有用，晚上的风太大了。

天上有大雁飞过去了。我以前见过这些大雁。它们让我更难过了。地上到处都是花，可是花都不好看了。我一个人看着窗户外面，什么时候天才能黑？外面下着小雨，一滴一滴的。我的难过，用一个"愁"字怎么能说得完呢？

这首词开头用了很多重复的词，读起来就好像一个人在那里叹气。

## 10. 浣溪沙 — 晏殊

**原文：**

一曲新词酒一杯，去年天气旧亭台。夕阳西下几时回？
无可奈何花落去，似曾相识燕归来。
小园香径独徘徊。

**生词：** 酒 = wine, 花 = flower, 太阳 = sun, 鸟 = bird (swallow)

**词人：** 晏殊是宋朝的大官，也是有名的词人。他写的词很优美。

**白话意思：**

我唱着一首新写的词，喝着一杯酒。今天的天气和去年一模一样。我坐的地方也是去年那个亭子。可是太阳下山了——它什么时候还会回来呢？

花落了，这是没有办法的事情，不能让它们不落。可是，有一些小鸟飞回来了，好像是去年见过的那些。我一个人在小花园的路上走来走去。

这首词在说时间。时间过去了就回不来了。花落了就不能再开了。可是鸟还会回来。有些东西会回来，有些不会。这让人有一点点难过。

"无可奈何花落去，似曾相识燕归来"是中国最有名的两句词之一。很多中国人都会说。
