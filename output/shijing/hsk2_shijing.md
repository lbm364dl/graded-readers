# 诗经 (The Book of Songs)

**HSK Level 2**

每首诗包括：原文、生词、故事背景、白话翻译。

---

## 1. 关雎

**原文：**

关关雎鸠，在河之洲。
窈窕淑女，君子好逑。

参差荇菜，左右流之。
窈窕淑女，寤寐求之。

**生词：** 河 = river, 女 = woman/girl, 好 = good/beautiful, 求 = pursue

**背景：** 这是诗经的第一首诗，也是最有名的一首。一个男人看到一个好看的女人，很喜欢她。

**白话意思：**

河里有两只鸟，它们在一起叫，声音很好听。有一个很好看的女人，一个好男人喜欢她。

河里有很多水草。那个好看的女人在河边。那个男人白天想她，晚上也想她。他想和她在一起，可是还没有在一起。他睡不着觉，想了一晚上。

这首诗写的是一个男人喜欢一个女人的感情。这种感情很真。

## 2. 蒹葭

**原文：**

蒹葭苍苍，白露为霜。
所谓伊人，在水一方。
溯洄从之，道阻且长。
溯游从之，宛在水中央。

**生词：** 白 = white, 水 = water, 路 = road, 长 = long

**背景：** 一个人想去找他喜欢的人，可是怎么也到不了那个人的地方。

**白话意思：**

河边的草又多又高。早上很冷，草上面有白白的东西。我喜欢的那个人，在河水的那一边。

我从下面走过去找她，路很难走，又远又长。我从上面走过去，她好像在河水的中间。

不管我怎么走，都到不了她的地方。她好像很近，又好像很远。这首诗写的是"想要可是得不到"的感情，让人觉得有一点难过。

## 3. 桃夭

**原文：**

桃之夭夭，灼灼其华。
之子于归，宜其室家。

桃之夭夭，有蕡其实。
之子于归，宜其家室。

**生词：** 花 = flower, 家 = home, 结婚 = marry

**背景：** 这是一首结婚的时候唱的歌。

**白话意思：**

桃花开了，红红的，非常好看！今天有一个女人要结婚了。她去了她的新家。她来了，这个家就会变得更好。

桃花不只是好看，还会长出果子。这个女人也是这样。她不只是好看，她还会让这个家更好，更有快乐。

在中国古代，桃花就是"好看的女人"和"快乐"的意思。这首诗到现在还常常在结婚的时候说。

## 4. 采薇

**原文：**

采薇采薇，薇亦作止。
曰归曰归，岁亦莫止。

昔我往矣，杨柳依依。
今我来思，雨雪霏霏。

**生词：** 回 = return, 以前 = before, 现在 = now

**背景：** 一个人去了很远的地方，很多年以后才能回家。

**白话意思：**

我在山上找野菜吃。我说要回家了，要回家了。可是一年又一年过去了，我还是不能回家。

以前我从家里走的时候，路边的树又绿又好看。现在我回来了，下着大雪，天很冷。

这首诗的最后两句特别有名。以前走的时候是春天，很美。现在回来了是冬天，下着大雪。人也老了，变了。回家的路很长，心里有高兴也有难过。

## 5. 硕鼠

**原文：**

硕鼠硕鼠，无食我黍。
三岁贯女，莫我肯顾。
逝将去女，适彼乐土。

**生词：** 大 = big, 吃 = eat, 走 = leave

**背景：** 老百姓不高兴，因为有人一直在吃他们的东西。

**白话意思：**

大老鼠啊大老鼠，不要再吃我们的东西了！三年了，我们给了你很多很多，你对我们一点也不好。我们不想在这里了，我们要走了。我们要去一个快乐的地方。

这首诗用了"大老鼠"来说那些不好的人。他们不工作，只是吃别人的东西。老百姓很生气，想要离开。这首诗是老百姓的声音。

## 6. 木瓜

**原文：**

投我以木瓜，报之以琼琚。
匪报也，永以为好也。

投我以木桃，报之以琼瑶。
匪报也，永以为好也。

**生词：** 给 = give, 送 = send/gift, 永 = forever

**背景：** 你给我一个东西，我也给你一个东西。这是朋友的感情。

**白话意思：**

你送给我一个水果，我送给你一个很好的东西。我不是在还你，我是想和你永远做好朋友。

你又送了我一个水果，我又送了你一个好东西。不是我要还你，是我想和你永远在一起。

这首诗说的是：真的朋友不是看东西多少。你对我好，我就想对你更好。你给我一个小东西，我想给你更多。这是心的感情。

## 7. 击鼓

**原文：**

击鼓其镗，踊跃用兵。
土国城漕，我独南行。

死生契阔，与子成说。
执子之手，与子偕老。

**生词：** 手 = hand, 老 = old

**背景：** 一个男人要去很远的地方，他和他爱的人说了一句话。

**白话意思：**

有人在打鼓，很多人要去远方。别人都去了一个地方，可是我一个人去了南边。

不管是生还是死，我和你说过的话不会变。我要拉着你的手，和你一起到老。

"执子之手，与子偕老"是这首诗最有名的两句话。意思是"我拉着你的手，和你在一起到老"。到现在，中国人结婚的时候还常常说这句话。可是这首诗其实很难过。因为这个人要走了，他不知道能不能回来。

## 8. 静女

**原文：**

静女其姝，俟我于城隅。
爱而不见，搔首踟蹰。

静女其娈，贻我彤管。
彤管有炜，说怿女美。

**生词：** 等 = wait, 红 = red

**背景：** 一个男人去见一个他喜欢的女人。

**白话意思：**

那个安静的女人真好看。她和我说，要在城边等我。我来了，可是她不出来。我看不到她，心里很着急，不知道怎么办。

后来她出来了，给了我一个红色的东西。这个红色的东西真好看。可是我觉得，她比这个东西更好看。

这首诗很有意思。那个女人是在和他玩，她在那里，可是不出来，让他着急。后来她出来了，还给了他一个东西。这首诗写的是两个年轻人见面的快乐。
